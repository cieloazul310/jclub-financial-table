Jクラブが過去に発表した決算概要や、増資・減資などの資本政策は、後から情報を把握しづらいという難点があります。というのも、Jクラブの過去のプレスリリースは、クラブ公式サイトのリニューアルなどによって閲覧できなくなることがしばしばあるためです。ネットニュースも同様に散逸しがちです。

ここでは、経営情報の**アーカイブ化**を目的とします。Jクラブの過去10年間の経営に関する情報を調べたいと思ったときに、時系列で「いつ」「何が起きたのか」「どのように報じられたのか」を簡単に把握できるようにすることを目指します。  
**速報性は重視せず**、過去の経営関連ニュースについても、確認でき次第さかのぼって更新していきます。

## 収集するもの

1. Jクラブの決算や予算に関するプレスリリース、報道
2. Jクラブの資本政策に関するプレスリリース、報道
3. Jリーグの配分金に関するプレスリリース、報道
4. Jリーグクラブライセンス制度の財務基準に関するプレスリリース、報道

## 収集しないもの

1. 選手・監督の契約更改や移籍金に関するプレスリリース、報道
2. Jクラブのスポンサー契約に関するプレスリリース、報道
3. Jリーグのスポンサー契約に関するプレスリリース、報道
4. 公益財団法人日本プロサッカーリーグほか、Jリーグ関連会社の経営に関するプレスリリース、報道

## 方針

1. 経営についての批評をしない  
2. Jリーグの規約・規程および会社法に準拠する  
3. 数字に基づいた推察は行う  
4. 一次ソースを明示する  

### 1. 経営についての批評をしない

本サイトにおける経営情報収集の目的は、あくまでアーカイブ化です。過去を振り返ったときに「いつ」「何があったのか」を把握できるようにすることを主眼としているため、経営に対する**批評は行いません**。  

Jクラブには、クラブごとに異なる経営哲学や置かれている状況があります。このクラブの経営はどうすべきであったか、といった評価や是非の判断は、各クラブのファンやステークホルダーがそれぞれ行うべきものであり、この場で一律に評することは不適当だと考えます。

また、自己資本比率や流動比率など財務諸表を基にした分析や、資金繰り、キャッシュ・フローに関する考察は行いません。資金面に関しては、できる限り報道や会見で明らかにされた内容を参照するにとどめます。

### 2. Jリーグの規約・規程や会社法に準拠する

Jリーグの規約・規程や、Jリーグクラブライセンス制度関連規程に抵触しないことは、各Jクラブにとって**最低限の行動指針**となります。  
そのため、経営に対する分析や批評は行いませんが、「○○年までに債務超過を解消しなければならない」といった、クラブライセンス制度を満たすために求められる条件については記述していきます。

- [Jリーグ 規約・規程](https://aboutj.jleague.jp/corporate/regulation/)
- [会社法](https://elaws.e-gov.go.jp/document?lawid=417AC0000000086)
- [財務諸表等の用語、様式及び作成方法に関する規則](https://elaws.e-gov.go.jp/document?lawid=338M50000040059_20210924_503M60000002061)

### 3. 数字に基づいた推察は行う

Jクラブが、すべての経営情報を詳細に開示するとは限りません。クラブからの発表がない場合でも、増資や減資などの資本政策が実施されているケースがあります。  

クラブからの公式発表がなくても、Jリーグが毎年公表する経営情報には数字の変化が表れるため、その数値に基づき、年度内にどのような資本政策が実施された可能性があるか、という推察は行います。

### 4. 一次ソースを明示する

クラブによるプレスリリースや報道を引用する際には、一次ソース（URL）を明示したうえで、著作権法第32条に定められた正当な範囲内で引用を行います。

[著作権が制限されるのはどんな場合？](https://www.cric.or.jp/qa/hajime/hajime7.html)

見出しなどで感情を煽るような強い表現は極力避け、経営情報や規約・規程への理解を深められるよう、数字に基づいて淡々と更新していきます。
