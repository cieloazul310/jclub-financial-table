**損益計算書**（P/L: Profit and Loss Statement）は、企業が一定期間（通常は1年間）に行った経済活動の内容と、その結果としての収支を示す財務諸表です。

## 損益計算書の見方

プロスポーツクラブの損益計算書を最も単純に見る方法は、**売上高**（**営業収入**）の規模を確認することです。クラブの売上高が大きいほど、言い換えれば**クラブの経営規模が大きいほど、サッカークラブとしての総合的な競争力が高い**と考えることができます。経営規模が大きいということは、選手や監督、スタッフに対して相対的に高い報酬を支払う能力を有していることを意味します。

一方で、一般企業の経営分析において重視されがちな**当期純利益**については、プロスポーツクラブの場合、その大小が必ずしも重要な指標になるとは限りません。純資産額に十分な余裕があり、単年の赤字が一時的なもので、かつ収支が管理可能な範囲に収まっている場合には、直ちに経営上の問題があるとは言えないためです。

## 損益計算書の項目

以下では、損益計算書の各項目について説明します。  
ここでは、Jクラブ個別経営情報開示資料を読み解くための簡易的な説明にとどめています。各項目の定義や詳細な取扱いについては、「[財務諸表等の用語、様式及び作成方法に関する規則](https://elaws.e-gov.go.jp/document?lawid=338M50000040059_20210924_503M60000002061)」第三章「損益計算書」を参照してください。

### 売上高（営業収入）

**売上高**（**営業収入**）とは、企業が本来の営業活動によって得た収入を指します。

プロスポーツクラブにおける営業収入の主な内訳は、スポンサー収入、入場料収入、Jリーグ配分金、物販収入、賞金、移籍金収入などです。このうち、**スポンサー収入**、**入場料収入**、**放映権料**（**リーグ配分金**）が営業収入の大部分を占めるのが一般的です。

内訳の詳細については、営業収入の項で説明しています。

### 営業費用

<Info title="営業費用 = 売上原価 + 販売費及び一般管理費" />

**営業費用**とは、営業活動を行うために発生した費用の総称です。

プロスポーツクラブの営業費用の内訳には、トップチーム人件費、移籍関連費用、試合関連経費、トップチーム運営経費、アカデミー関連経費、女子チーム運営経費、物販関連費、販売費および一般管理費などが含まれます。クラブによって差はありますが、営業費用のおよそ半分前後が、選手や監督の年俸を含む**チーム人件費**に充てられるケースが多く見られます。

内訳の詳細については、営業費用の項で説明しています。

### 営業利益

<Info title="営業利益 = 売上高 -（売上原価 + 販売費及び一般管理費）" />

<Info title="営業利益 = 営業収入 - 営業費用" />

営業利益は、営業収入から営業費用を差し引いた金額です。クラブの**本業での収支状況**を示す指標と位置づけることができます。

### 営業外収益

営業外収益とは、先に述べた営業活動以外から生じた収益を指します。プロスポーツクラブにおける営業外収益の主な内容は、受取利息、寄付金収入、補助金収入、リーグ支援金（配分金ではない）などです。

一定規模の営業外収益が恒常的に計上されているクラブもありますが、これはクラブ後援会や関係者からの継続的な支援によるものと考えられます。

### 営業外費用

営業外費用とは、営業活動以外に起因して発生した費用です。具体的には、支払利息、株式交付費、為替差損などが該当します。

また、選手やスタッフとの契約解除に伴って発生する違約金についても、営業外費用として計上されるケースがあります。ただし、契約解除金の扱いはクラブごとに異なり、**特別損失として計上される場合もある**点には留意が必要です。

### 経常利益

<Info title="経常利益 = 営業利益 +（営業外収益 - 営業外費用）" />

経常利益は、営業利益に営業外収益と営業外費用を加味した金額で、クラブの**経常的な収支状況**を示します。

### 特別利益

特別利益は、クラブの通常の経済活動とは直接関係しない、一時的に発生した収益です。寄付金収入や、スポンサー料とは別枠で行われる親会社からの損失補填などが該当します。

### 特別損失

特別損失は、通常の経済活動とは関係なく、一時的に発生した費用です。契約金償却損、固定資産除却損、監督の契約解除金などが含まれます。

### 当期純利益

<Info title="税引前当期利益 = 経常利益 +（特別利益 - 特別損失）" />

<Info title="当期純利益 = 税引前当期利益 - 法人税および住民税等" />

当期純利益は、クラブが1年間の経済活動を通じて最終的に得た収支を示します。当期純利益がプラスの場合を黒字、マイナスの場合を赤字と呼びます。

2012年に導入されたJリーグクラブライセンス制度では、クラブ経営の健全化を目的として「**3期連続赤字の禁止**」が定められました。ただし、2018年の交付規則改正により、**純資産額に十分な余裕がある場合には、3期以上連続の赤字を認める**という規制緩和が行われています。この緩和措置は、2024年現在、J1・J2クラブライセンス判定には適用されており、J3クラブライセンス判定には適用されていません。

当期純利益は一般企業にとっては重要な「結果指標」ですが、プロスポーツクラブでは、収支が均衡するように計画的な経営が行われるため、極端な数値でない限り、単年度の当期純利益のみをもって経営を評価することは適切とは言えません。そのため、当期純利益のクラブ間比較も、限定的な意味しか持たないと考えられます。

ただし、単年の赤字自体が問題とならない場合であっても、赤字が数年にわたって累積すると、負債が資産を上回る**債務超過**に陥る可能性があります。Jリーグクラブライセンス制度では、**債務超過のクラブにはクラブライセンスが交付されません**。

このため、プロスポーツクラブの経営情報を読む際には、当期純利益を単独で見るのではなく、**経営規模や純資産額と併せて確認する**ことが重要です。  
例えば、前期の純資産額が5億円程度あるクラブにとって、単年で生じた5,000万円の赤字は、直ちに経営上の問題とは言えません。一方で、前期の純資産額が1億円未満のクラブが同規模の赤字を計上した場合、次期以降に債務超過に陥るリスクを高めることになり、望ましい状態とは言いにくいでしょう。

<Alert title="財務基準の特例措置について">
  新型コロナウイルスの流行に伴い、一定期間、債務超過の禁止および「3期連続赤字の禁止」をクラブライセンス判定の対象としない**特例措置**が取られています。また、2023年シーズンの均等配分金の減額に伴い、2023年度決算も特例措置の対象となることが決定しています。

<ins>
  さらに、2026年夏に実施されるシーズン移行に伴い、26特別シーズンおよび26/27シーズンに対応する事業年度が特例措置の対象となり、27/28シーズンに対応する事業年度が猶予期間の対象となることが、2025年秋に決定しています。このため、「3期連続赤字の禁止」は27/28シーズン対応事業年度を1期目としてカウントが開始され、29/30シーズン中に実施される30/31シーズンのクラブライセンス審査において、再び判定対象となります。
</ins>
</Alert>

### 関連する法人の営業収益

関連する法人（アカデミーなど、サッカーおよびその他関連事業を運営する法人）の営業収益は、別法人の収益であるため**損益計算書には含まれません**が、2018年度の経営情報開示から**参考情報として**掲載されるようになりました。

C大阪や湘南などでは、アカデミーや普及活動、総合型スポーツクラブ事業を一般社団法人や公益財団法人が運営しています。クラブグループ全体という観点で見れば、運営法人の営業収入に関連法人の営業収益を加えた金額を、**実質的なクラブ規模の目安**として捉えることができます。

ただし、運営法人と関連法人との間で行われている内部取引については相殺消去が行われていないため、単純に加算した数値が正確な連結売上高を示すものではない点には注意が必要です。

<Details summary="関連する法人の営業収益についての説明" title="2018年度 クラブ経営情報開示 メディア説明会 発言録" url="https://www.jleague.jp/news/article/15296/" asBlockquote>
  そのほか、『関連する法人（アカデミーなどサッカー及びその他関連する事業を運営する法人）』の営業収益という項目が増えています。クラブを運営し、トップチームが所属している運営法人ではなく、それ以外の組織でアカデミーを運営している場合です。このケースもライセンス上は認められており、関連法人でもアカデミーを保有していれば、カウントするとしています。ただ、アカデミーを他の法人に移して、本体の財務状況を良いように見せていないのかという懸念もライセンス的にはありますので、クラブライセンス上は、これまでも関連する法人もチェックの対象としておりました。これまでは関連法人の情報は開示の対象としていなかったのですが、近年、NPO法人など、こうした形でアカデミーを運営するスタイルが増えてきていますので、開示項目として増やしたらどうかということをクラブと話し合い、合意ができましたので開示いたしました。  
  札幌は、本体が29億8800万円の営業収益です。そのうちのアカデミー関連収入が2500万円となっておりますが、実態としてはこれ以外にも関連法人で売上が2億4000万円あったことがわかります。そのためサッカー関連ビジネスを運営する法人全体で見ると、29億8800万円+2億4000万円の合算で推し量ることができます。  
  ただし、注意点として、「クラブと関連法人間の内部取引の消去は行っていない」ということです。これは、例えば、札幌が関連法人にコーチを派遣すると、業務委託料が札幌へ支払われるので収入になります（関連法人側では経費）。こうした内部取引については、相殺消去をしていないので、内部取引を消去した上でグループ全体での売上高や利益がどうかという指標には使いにくい数字と認識ください。内部取引消去を正確に行っていないまでも、おおよそのイメージとして、一つの目安としてみていただきたく、すべてのPLをつまびらかにできていませんが、ある程度参考情報として確認いただけたらと思います。

更に、具体例としてはＣ大阪をあげますが、Ｃ大阪のようにトップチームを運営しているのは株式会社で、アカデミー等を運営していない場合があります。Ｃ大阪は日本でも有数のアカデミー組織を持っていますが、これまでの経営情報開示では、別の組織で運営している情報については開示しておらず、アカデミー関連収入が０で表示されていました。これまで、それはおかしいのではという声が聞かれていましたが、「関連する法人で行っている」と、説明しておりました。ですが、今回開示した情報を見ていただくと、Ｃ大阪は、約２２億円、関連する法人での営業収益があることがわかります。  
 このように一つの目安としてみていただきたいと考えますので、ある程度参考情報として確認いただけたらと思います。Ｊ２では山形や町田もアカデミー関連は０でしたので、関連する法人で売上があったことがわかります。Ｊ３では群馬、相模原、鹿児島が該当します。  
 また、参考時の注意点としては、必ずしもスクール事業や育成組織の売上だけではなく、指定管理収入も含まれています。Ｃ大阪の金額が高いのは、そのためで、その他のサッカービジネスを運用する上で派生する営業収益もこちらに含まれていますので、今回はそのくくりで開示させていただいている情報だと認識ください。

</Details>
