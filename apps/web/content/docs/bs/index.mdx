**貸借対照表**（B/S: Balance Sheet）は、**企業の財務状況を一定時点で示す財務諸表**です。  
損益計算書が**一定期間（通常は1事業年度）における経営成績**を表すのに対し、貸借対照表は**決算時点における企業の財務状態や経営の安定度**を示します。

貸借対照表は決算日時点の数値で構成されるため、**年度表記と実際の財務状況の時点が一致しない**点には注意が必要です。  
例えば1月決算の企業の場合、2020年度決算の貸借対照表は、**2021年1月末時点**の財務状況を表しています。年度表記と実際の時点に1年弱のズレが生じるため、他の資料と比較する際には留意する必要があります。

## 貸借対照表の見方

プロスポーツクラブの貸借対照表を最も単純に読み解く方法は、**純資産額**（**純資産の部**）に注目することです。  
**純資産額が大きいほど、財務的な余力があり、経営が安定しているクラブである**と評価できます。

一方で、負債が資産を上回り、**純資産がマイナスとなっている状態**は**債務超過**と呼ばれます。  
Jリーグではクラブライセンス制度により、原則として**債務超過の状態にあるクラブへのライセンス交付を認めていません**。

また、**資本金の増減**にも注目すると、クラブの経営環境を読み解く手がかりになります。  
資本金の変化は、**増資や減資が年度内に実施された**ことを意味します。増資は資金調達手段であると同時に、**株主構成や経営権の変化**につながる場合があります。

すべてのJクラブが株主構成を詳細に公表しているわけではありませんが、当該年度のプレスリリースや報道内容と照らし合わせることで、**経営体制の変化や資本政策の意図を推測することは可能**です。

なお、Jリーグが公表する経営情報は、**あくまで1事業年度単位の決算情報**です。  
このため、**月次の資金繰りや短期的な資金不足の有無**といった点までは把握できないことを前提に読む必要があります。

## 貸借対照表の項目

以下では、貸借対照表の各項目について説明します。  
Jクラブの「個別経営情報開示資料」を読み解くための**簡易的な解説**であり、厳密な会計定義については、法令に基づく資料を参照してください。

<Info title="資産の部 = 負債の部 + 純資産の部" />

貸借対照表では、**資産の部が借方**、**負債の部および純資産の部が貸方**に配置されます。  
借方と貸方の合計額は必ず一致しなければならず、負債の部が資産の部を上回る場合、その差額として純資産の部がマイナスになります。この状態が**債務超過**です。

### 資産の部（総資産）

<Info title="資産の部 = 流動資産 + 固定資産" />

資産の部（総資産）は、**企業が保有する経済的価値の合計**を示します。  
資産は性質に応じて、**流動資産**と**固定資産等**に分類されます。

#### 流動資産

流動資産は、**1年以内に現金化される、または消費される資産**です。  
主な内訳として、現金及び預金、売掛金、商品、貯蔵品、前払費用、未収入金などが含まれます。

#### 固定資産等

固定資産等は、**長期間にわたり継続的に使用される資産**です。

- 有形固定資産（建物、構築物、車両運搬具、工具器具及び備品など）  
- 無形固定資産（借地権、**選手登録権**、商標権、ソフトウェア、電話加入金など）  
- 投資その他の資産（出資金、長期前払費用、敷金保証金など）

プロスポーツクラブ特有の会計処理として、**移籍金を支払って獲得した選手の登録権を無形固定資産として計上する**点が挙げられます。

例えば、2020年冬に移籍金1億円で選手を獲得し、5年契約を締結した場合、2020年度決算（2021年1月期）では、**選手登録権を取得原価である1億円で無形固定資産として計上**します。

この時点で移籍金の支払いが完了していれば、流動資産（現金・預金）が1億円減少します。  
未払いの場合は、流動負債（未払金）が1億円計上されます。分割払いの場合も、未払部分は同様に負債として計上されます。  
なお、この段階では**移籍金は費用としては計上されていません**。

選手登録権は、契約期間にわたって**減価償却費として費用配分**されます。  
この例では、2021年度から2025年度までの5年間、**毎年度2,000万円ずつ移籍関連費用として計上**され、無形固定資産の帳簿価額は年々減少し、契約満了時点でゼロになります。

### 負債の部（総負債）

<Info title="負債の部 = 流動負債 + 固定負債" />

負債の部（総負債）は、**企業が将来返済または支払いを行う義務を負っている経済的価値の合計**です。  
流動負債と固定負債に分類されます。

#### 流動負債

流動負債は、**1年以内に支払期限が到来する負債**です。  
買掛金、短期借入金、1年以内返済予定の長期借入金、未払金、未払費用、未払法人税等、前受金、預り金、前受収益などが含まれます。

#### 固定負債

固定負債は、**返済期限が1年を超える負債**です。  
社債、長期借入金、退職給付引当金などが該当します。

### 純資産の部（純資産額）

<Info title="純資産の部 = 資産の部 - 負債の部" />

<Info title="純資産の部 = 資本金 + 資本剰余金等 + 利益剰余金" />

純資産の部（純資産額）は、資産の部（総資産）から負債の部（総負債）を差し引いた値であり、**企業に帰属する正味の財産**を表します。  
内訳として、資本金、資本剰余金等、利益剰余金の合計と一致します。

資本金および資本剰余金等は、年度内に増資や減資が行われない限り原則として変動しません。  
このため、**純資産額の増減は、基本的に利益剰余金の増減、すなわち当期純利益の結果を反映したもの**となります。

**純資産がマイナスの状態**が**債務超過**です。  
これは、企業が保有するすべての資産を売却しても、負債を返済しきれない状態を意味します。

Jリーグのクラブライセンス制度では、通常、**債務超過に陥ったクラブにはライセンスが交付されません**。  
この制度は2012年度に施行され、財務基準における「債務超過の禁止」は、2015シーズンのクラブライセンス判定（2014年度決算）から本格的に審査対象となりました。

2013年度決算ではJ1・J2合計で12クラブが債務超過でしたが、2014年度決算ではJ1・J2で0クラブとなりました。  
さらに、J3にも債務超過禁止が適用された2015年度以降は、J1からJ3までの全クラブが一度は債務超過を解消しています。

<Alert title="新型コロナウイルスの影響による特例措置">
新型コロナウイルス感染症の影響により、2020年度決算では10クラブ、2021年度決算では12クラブが債務超過となりました。ただし、クラブライセンス制度の財務基準に特例措置が設けられ、2024年度決算までは猶予付きで債務超過が認められています。2025年度決算からは、再び原則どおり債務超過が禁止される予定です。
</Alert>

#### 資本金

資本金は、株式会社の**元本に相当する会計上の金額**です。  
通常は一定ですが、年度内に**増資**または**減資**が行われた場合に変動します。

企業は出資を受ける際、出資額に応じて**株式**を発行します。  
株式は取締役会における**議決権**を伴い、取締役や役員の選任・解任は、原則として議決権の**過半数**で決定されます。

そのため、発行済株式の過半数を保有する（または過半数を持たずとも支配的影響力を持つ）出資者は、企業の場合は**親会社**、個人の場合は**オーナー**と位置づけられます。

クラブライセンス制度では債務超過が原則認められていないため、**債務超過回避の手段として増資が行われるケース**が一定数見られます。

<Alert>
資本金および資本剰余金等は決算時点の数値であり、年度内に行われたすべての増資・減資の経緯を反映するものではありません。クラブの公式リリースや報道と併せて確認することが重要です。
</Alert>

#### 資本剰余金等

資本剰余金等は、**資本準備金とその他資本剰余金の合計**です。  
実務上は、ほぼ資本準備金と同義として扱われることが一般的です。

資本金と同様、原則として一定の数値ですが、増資や減資が行われた場合にのみ変動します。  
増資時には、払込金額の**2分の1を上限として資本準備金に計上**することができます（会社法445条）。  
また、減資時には、減少した資本金を資本準備金に振り替えることも可能です。

#### 利益剰余金

利益剰余金は、**過去の当期純利益の累積額**です。  
マイナスの場合は、**累積赤字**と表現されることがあります。

原則として、当年度の利益剰余金は、前年度の利益剰余金に当年度の当期純利益を加えた値になります。  
この関係が成り立たない場合、年度内に**減資によって資本金を利益剰余金へ振り替え、累積赤字の解消を図った**可能性が考えられます。

なお、減資による資本金の利益剰余金への振り替えは、**利益剰余金がマイナスの場合に限られ、振り替え後の利益剰余金は0を上限**として認められています。
