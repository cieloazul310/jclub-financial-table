企業が資本金を増やすことを**増資**、資本金を減らすことを**減資**と言います。  
原則として、増資または減資の実施には株主総会の決議が必要となります。

## 増資

資本金を増やすことを**増資**と言います。  
出資を受ける企業は、出資者に対し、出資額に応じた**株式**を発行します。出資者が保有する株式数は、その企業における**議決権の割合**を構成します。

取締役や監査役等の選任・解任は、株主総会における議決権の**過半数**によって決定されるため、発行済株式の過半数を保有する出資者は、経営に対して決定的な影響力を持ちます。このような出資者を、法人である場合は**親会社**、個人である場合は**オーナー**と呼ぶのが一般的です。なお、必ずしも過半数を保有していなくとも、実質的に経営を支配していると認められる場合もあります。

増資には**有償増資**と**無償増資**があり、有償増資は以下の3つに分類されます。Jクラブにおける増資は、多くの場合、特定の出資者を引受先とする**第三者割当増資**が用いられています。

- 株主割当増資  
  既存株主が、持株比率に応じて出資を行う方法。出資後も持株比率は変化しない。
- **第三者割当増資**  
  既存の持株比率に関わらず、特定の第三者に対して株式を発行する方法。出資後の持株比率は変化する。
- 公募増資  
  不特定多数の一般投資家を対象に株主を募集する方法。

増資を実施すると発行済株式総数が増加するため、第三者割当増資の引受先がすべて新規株主である場合、既存株主の持株比率は低下します。このため、増資の実施には既存株主の理解と合意が不可欠となります。

なお、増資の払込金額のうち**2分の1を上限として**、資本金ではなく**資本準備金**に計上することが可能です（会社法第445条第2項）。  
例えば、普通株式を1株5万円で100株発行した場合、払込金額の半額を資本準備金に充当すると、資本金は250万円、資本剰余金等（資本準備金）は250万円となります。

### 増資の例

| **年**     |        **純資産** |       **資本金** | **資本剰余金等** |   **利益剰余金** | **当期純利益** |
| ---------- | ----------------: | ---------------: | ---------------: | ---------------: | -------------: |
| **2017**   |                46 |              167 |              120 |            \-241 |             24 |
| **増資後** |              1194 |              741 |              694 |            \-241 |                |
| **2018**   |              1161 |              741 |              694 |            \-274 |           \-33 |
| **前年差** | <Diff>1115</Diff> | <Diff>574</Diff> | <Diff>574</Diff> | <Diff>-33</Diff> |                |

この例では、2018年度内に実施された増資により、資本金は1億6700万円から7億4100万円へ、資本剰余金等は1億2000万円から6億9400万円へと、それぞれ5億7400万円ずつ増加しています。  
その結果、2018年度の純資産額の増加分は、資本金および資本剰余金等の増加額合計11億4800万円から、利益剰余金の減少分を差し引いた**11億1500万円**となります。

このように、従来の資本金を大幅に上回る規模の増資が行われている場合、親会社やオーナーの変更など、クラブの経営体制に大きな変化が生じている可能性があります。

### 増資の可能性

増資の実施について、クラブが必ずしもプレスリリースを公表するとは限りません。  
Jクラブ経営情報開示の貸借対照表において、以下のような特徴が見られる場合、増資が行われている可能性があります。

- 資本金および資本剰余金等が増加している場合

## 減資

資本金を減らすことを**減資**と言います。  
株主に対して金銭を返還する**有償減資**は例外的であり、実務上は、資本金の減少分を資本剰余金等または利益剰余金（欠損補填の場合）へ振り替える**無償減資**が大半を占めます。  
**無償減資の前後で、純資産の総額は変化しません。**

### 減資の例（資本剰余金等への振替）

| **年**     |     **純資産** |        **資本金** | **資本剰余金等** | **利益剰余金** | **当期純利益** |
| ---------- | -------------: | ----------------: | ---------------: | -------------: | -------------: |
| **2018**   |           1161 |               741 |              694 |          \-274 |           \-33 |
| **減資後** |           1161 |               100 |             1335 |          \-274 |                |
| **2019**   |           1168 |               100 |             1335 |          \-267 |              7 |
| **前年差** | <Diff>7</Diff> | <Diff>-641</Diff> | <Diff>641</Diff> | <Diff>7</Diff> |                |

この例では、2019年度内に実施された減資により、資本金は7億4100万円から1億円へと減少し、減少分6億4100万円が全額資本剰余金等へ振り替えられています。  
その結果、2019年度決算における純資産額の増減は、当期純利益7百万円と一致しています。

なお、この例では減資後の資本金を1億円としています。決算日時点の資本金が1億円以下である場合、租税特別措置法第42条に定める法人税率の特例を適用できる可能性があります。

### 減資の可能性

減資の実施についても、クラブが必ずしもプレスリリースを行うとは限りません。  
Jクラブ経営情報開示の貸借対照表において、以下のような特徴が見られる場合、減資が行われている可能性があります。

- 資本剰余金等の変化量が、資本金の変化量より大きい場合
- 利益剰余金の変化量が、当期純利益と一致しない場合

## 増資および減資、減資および増資

同一事業年度内に、増資と減資の両方を実施するケースも存在します。  
増資を行った後に減資を実施する場合もあれば、先に減資を行った上で増資を実施する場合もあります。

### 増資および減資の例

|            |       **純資産** |        **資本金** | **資本剰余金等** |   **利益剰余金** | **当期純利益** |
| ---------- | ---------------: | ----------------: | ---------------: | ---------------: | -------------: |
| **2020**   |               16 |               354 |              125 |            \-463 |              9 |
| **増資後** |              416 |               554 |              325 |            \-463 |                |
| **減資後** |              416 |               100 |              325 |              \-9 |                |
| **2021**   |              348 |               100 |              325 |             \-77 |           \-68 |
| **前年差** | <Diff>332</Diff> | <Diff>-254</Diff> | <Diff>200</Diff> | <Diff>386</Diff> |                |

この例では、2021年度内に増資および減資が実施されています。  
増資によって資本金および資本剰余金等がそれぞれ2億円増加した後、減資によって資本金の減少分4億5400万円が全額利益剰余金へ振り替えられています。

その結果、2021年度決算時点の資本金は1億円、資本剰余金等は3億2500万円、利益剰余金は減資後のマイナス900万円に当期純損失を加えた**マイナス7700万円**となります。

### 増資および減資、または減資および増資の可能性

これらの手続についても、クラブが必ずしもプレスリリースを公表するとは限りません。  
Jクラブ経営情報開示の貸借対照表において、以下の特徴が見られる場合、増資および減資、または減資および増資が行われている可能性があります。

- 資本金の減少額が、資本剰余金等および利益剰余金の増加額から当期純利益を控除した金額と一致しない場合
