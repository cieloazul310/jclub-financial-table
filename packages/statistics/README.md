# Jクラブ経営情報統計パッケージ

> [!NOTE]
> このパッケージはコアパッケージに含まれています

> [!WARNING]
> このパッケージは今後仕様が変更になる可能性が高いです

## 内容

- Jクラブ経営情報ポータルに搭載している60+3クラブの情報を搭載
- Jクラブ経営情報ポータルで表示できる年度の情報を搭載
- Jクラブ経営情報ポータルで表示しているデータのキーと日本語表記の対応表を搭載

## インストール

```sh
npm install @cieloazul310/jclub-financial-statistics
```
